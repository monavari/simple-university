# How to use git and github

change files -> commit to local repo with message -> push to github repo (upstream)

1. after chanaging files you have to commit the changes to local repository
2. to commit changes from VSCODE IDE use the left icon Ctrl+Shift+G
3. for each commit you need message 
4. write the message in the text box and click commit. Message should be meanningfull
5. after commitnig the chagnes are stored in local repository not github
6. To apply changes to github repository [upstream] you have to push the changes to gihub by clickng the Push icon (upward arrow)
7. you can check the changes on github
